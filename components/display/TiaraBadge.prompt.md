Compact status / count label with solid or light fill, optional status dot.

```jsx
<TiaraBadge text="Confermato" color="green" />
<TiaraBadge text="In preparazione" type="light" color="amber" />
<TiaraBadge mode="dot-word" color="blue" text="B2B" />
<TiaraBadge mode="number" count={12} />
```

`mode`: `word` `number` `dot` `dot-word`. `type`: `solid` `light`. `color`: accent, neutral, red, orange, amber, green, blue, purple, pink (+ dark- variants).
